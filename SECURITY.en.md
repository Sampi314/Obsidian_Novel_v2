# 🛡️ Security Policy

This is a personal project focused on the creation of literary content (Xianxia). While it does not contain complex application source code, we are committed to maintaining the integrity and safety of the data within the repository.

## 📞 Reporting a Vulnerability

If you discover any security-related issues (e.g., exposure of personal information in Markdown files, automated scripts with unusual behavior), please follow these steps:

1.  **DO NOT** open a public Issue on GitHub to avoid unnecessary risk.
2.  Send a detailed report directly to the administrator via email: **nkhoihue@gmail.com**.
3.  We will respond and address the issue as soon as possible.

## ⚠️ Automated Script Warning

The repository contains maintenance scripts (`scripts/`) written in Python and Node.js.
*   These scripts are designed for an internal development environment.
*   External users should carefully inspect the script content before executing them on their personal systems.
*   Only use scripts provided officially in the `main` branch of this repository.

Thank you for your interest in the project's safety!
